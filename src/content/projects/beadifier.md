## About

Beadifier is a specialized web application that converts any image into bead art patterns, designed for crafters and pixel art enthusiasts.

## Key Features

- **Multiple Dithering Algorithms**: Choose from Floyd-Steinberg, Ordered, Atkinson, and more
- **Color Palette Management**: Support for official Hama and Perler bead colors
- **Size Optimization**: Smart resizing with aspect ratio preservation
- **Export Options**: Download patterns as PDF, PNG, or printable instructions

## Technical Highlights

- Pure client-side processing - no server uploads needed
- WebWorkers for heavy image processing
- Canvas API for pixel-perfect manipulation
- Responsive design works on desktop and mobile

## Community Impact

Used by 5,000+ crafters worldwide to create stunning bead art. Featured in several maker communities and craft blogs.
